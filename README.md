# qrshibie 二维码批量识别一键复制

## 🚀 功能特点
- 📱 可离线便捷使用
- 🔐 安全后台无记录
- 📱 可批量识别图片
- 📱 可一键复制结果
- 📱 多平台可以适用

## 🛠 离线版本/在线版本
- 离线版本文件 qr.html + jsQR.js
- 在线版本文件 qr-online.html

## 使用说明（以宝塔面板为例）
- 复制版本文件到网站根目录下或指定目录即可使用
- 如果需要主页访问将html前缀名改成index
- 离线版本 打开网页后 可以断网使用 安全可靠
